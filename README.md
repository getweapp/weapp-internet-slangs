# 微信小程序－新词 Get!

### 说明：

实现了排版美观的网络用语科普与查询功能，特色：
- 拼音，汉字和图片的排版
- 详情页支持显示图片、音频、视频的多媒体附件

### 数据接口:

使用本地数据

### 目录结构：

- images — 存放项目图片文件
- pages — 存放项目页面渲染相关文件
- utils — 存放搜索功能代码

### 开发环境：

微信web开发者工具 v0.11.112301
